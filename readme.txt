Provides a set of basic registration and login shortcode through the front-end of your WordPress site.

== Description ==

Use the login, registration, password recovery and password change shortcodes so that your users can update the registration using a front-end page.
Ideal for use on member sites such as associations, clubs, cooperatives, unions, schools, churches, etc. or any organization or group that needs to provide access via login and password.

== Installation ==

After accessing your site in admin mode, click on the "Plugins" tab, then on "add new", in the search type "fixlogin". When the plugin appears, click "install" and then "Activate"

== Common questions ==

= Can I use any page to serve as a login or registration? =

Yes. Just paste in the proper shortcode.
